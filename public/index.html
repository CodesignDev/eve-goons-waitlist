Goonfleet's Incursion Squad Waitlist

Login via oAuth to the right. We request several scopes to gather data about your experience and manage fleets via the web UI.

If you have any questions, hit up Makeshift Storque on Jabber

Scope information:

  • Location: So we can track where parts of the fleet are
  • Ship Type: Track fleet composition
  • Read Skills: Make sure you have recommended minimums
  • Read Skillqueue: Or you will do soon
  • Read Wallet: So we can see who isn't in site with us, as well as payout stats
  • Read Killmails: Hah you died
  • Read Assets: Allows us to see your ship fits
  • Read Fleet: If you're an FC, info about the fleet
  • Write Fleet: If you're an FC, can edit the fleet from the UI
  • Write Waypoint: Useful for FC's to set destinations
  • Read Fitting: See saved fits
  • Read Online: Alert the FC if you drop connection
  • Read Implants: Check if you have suggested implants
  • Read Fatique: Useful for the FC to know
